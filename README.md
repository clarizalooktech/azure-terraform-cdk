# Azure Terraform CDK - Create Virtual Machine (Typescript)
Repo for creating an Azure VM resource using infrastructure as code - Terraform CDK

## 1. Install Tf CLI
> npm install --global cdktf-cli@latest

## 2. Synthesize Tf
> cdktf synth

## 3.  Deploy tf
> cdktf deploy

## 4.  Destroy resources
> cdktf destroy
